MAT
----
A light-weight C++ linear algebra template library. Made to be compatable with the arduino.

Matricies are passed as one dimensional arrays.

Edit config.hpp to match your platform.

Example:

	float matid[9] = {
		1, 0, 0,
		0, 1, 0
		0, 0, 1,
	};
